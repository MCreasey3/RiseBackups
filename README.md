# RiseBackups
Manual save file backups for Steam games


I'm using this PowerShell script to manually back up my save files for Monster Hunter Rise. Playing the game with mods installed can sometimes cause Steam to corrupt and subsequently lose all Cloud saved data. 

游리 When setting origin folder, replace "useridhere" with your Steam user ID visible in the directory. 游리

游리 Destination folder for backups can be located anywhere, just make sure you've typed it correctly. 游리

游댯 maxFileCount can be adjusted based on preference. Oldest backup folder is set to autodelete once it's capped. To my knowledge, 100 saves is roughly ~1GB of space. 游댯
